# Store Submission

Complete checklist for App Store and Play Store submission.

## App Store Checklist

- [ ] App screenshots (all required sizes)
- [ ] App icon (1024x1024)
- [ ] App preview video (optional)
- [ ] App description
- [ ] Keywords for ASO
- [ ] Privacy policy URL
- [ ] Support URL
- [ ] Age rating
- [ ] Build uploaded
- [ ] App reviewed and approved
- [ ] Release date set

## Play Store Checklist

- [ ] App screenshots (phone, tablet)
- [ ] Feature graphic (1024x500)
- [ ] App icon (512x512)
- [ ] Short description (80 chars)
- [ ] Full description (4000 chars)
- [ ] Privacy policy URL
- [ ] Content rating questionnaire
- [ ] AAB uploaded
- [ ] Pricing set
- [ ] Distribution countries selected

## Common Rejection Reasons

### iOS
- Missing privacy policy
- Incomplete app information
- Crashes on launch
- Broken features
- Missing permissions descriptions

### Android
- Misleading app description
- Broken deep links
- Missing privacy policy
- Content rating mismatch
