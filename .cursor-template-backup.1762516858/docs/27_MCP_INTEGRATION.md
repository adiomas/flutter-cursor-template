# MCP Integration

Integrate Model Context Protocol (MCP) tools for enhanced development workflow.

## MCP Context7

Access library documentation on-demand.

### Resolve Library

```
Find library documentation for "riverpod"
```

### Get Documentation

```
Get detailed documentation for specific topics in Riverpod
```

## MCP Supabase

Interact with Supabase directly from development environment.

### Search Documentation

```graphql
{
  searchDocs(query: "authentication", limit: 5) {
    nodes {
      title
      href
      content
    }
  }
}
```

### Database Operations

- List tables
- Execute SQL queries
- View migrations
- Check RLS policies

## MCP Browser Tools

Debug web applications:
- Get console logs
- Check network errors
- Take screenshots
- Run accessibility audits
- Run performance audits

## MCP Firecrawl

Scrape web content for research:

```
Scrape documentation from official Flutter website
```

## Workflow Integration

### Documentation Lookup
1. Need to understand a package
2. Use MCP Context7 to resolve library ID
3. Get focused documentation
4. Implement feature

### Database Development
1. Design schema
2. Use MCP Supabase to create tables
3. Test queries directly
4. Implement repository

### Web Debugging
1. Run web app
2. Use MCP Browser Tools to check console
3. Analyze network requests
4. Fix issues
